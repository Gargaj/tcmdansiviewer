ANSI Viewer Lister plugin for Total Commander
=================================================

Coded by Gargaj / Conspiracy (http://gargaj.umlaut.hu)

New versions at:
  https://github.com/Gargaj/tcmdansiviewer/releases

Sourcecode at:
  https://github.com/Gargaj/tcmdansiviewer

Test material:
- Lord Maxxtro's Artpack #16
- Arv Terror Corps - 100 lines o' crap
- Various other ACiD stuff
- UE ANSI pack (http://ue.untergrund.net/ansi/ALLANSI.ZIP)
- Blocktronics packs (http://pc.textmod.es/pack/blocktronics_block_to_the_future/)
(Basically whatever I come across.)

Comparison basis was ACiDDraw v1.25r and PNG renders

Uses stb_image_write.h by Sean Barrett et al:
  https://github.com/nothings/stb/blob/master/stb_image_write.h



Keys
----------------
  F12         - dump picture to image file
  numpad +/-  - zoom in/out
